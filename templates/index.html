Index

  • Fighters Allows users to look up an existing fighter, modify an existing fighter (change weapon, change name), and create a new fighter. Fighter profiles for existing fighters will link to that fighter's prize results.
  • Weapons Allows users to look up an existing weapon, modify an existing weapon (change weapon, change name), and links to fighter using that weapon (if any).
  • Results Contains leaderboard (Fighters with most wins) and allows users to look up a Fighter's Prizes
  • Fights Allows lookup and creation/modification of fights (who is participating, when the Fight is). Allows assignment of Prizes to Fights and assignment of Fight winners.
