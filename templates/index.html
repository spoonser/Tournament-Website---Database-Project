Index

  • Fighters Allows users to look up an existing Fighter or create a new Fighter. Fighter profiles for existing Fighters will link to that Fighter's Prize results.

  • Fighter Wins Contains leaderboard (Fighters with most wins) and allows users to look up a Fighter's Prizes.
